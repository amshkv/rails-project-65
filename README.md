## Requirements

- Ruby 3.2.2
- NodeJS 16 and later
- Yarn
- Gitmoji CLI

## Setup

```sh
make setup
```

## Run

```sh
make start
```

## Development

### Install hooks

```sh
make install-hooks
```

### Tests

```sh
make test
```

### Linter

```sh
make lint
```
